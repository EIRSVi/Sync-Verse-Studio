## Typography

### Font Hierarchy
```
Level 1: Dashboard Title
• Font: Segoe UI, 22F, Bold
• Color: #212121
• Usage: Main dashboard heading

Level 2: Section Title
• Font: Segoe UI, 18F, Bold
• Color: #212121
• Usage: User Management, Product Management titles

Level 3: Card Title
• Font: Segoe UI, 11F, Bold
• Color: White
• Usage: Metric card titles

Level 4: Grid Header
• Font: Segoe UI, 11F, Bold
• Color: White
• Usage: Data grid column headers

Level 5: Grid Content
• Font: Segoe UI, 10F, Regular
• Color: #212121
• Usage: Data grid cell content

Level 6: Buttons
• Font: Segoe UI, 10F, Bold
• Color: White
• Usage: All action buttons

Level 7: Metadata
• Font: Segoe UI, 9F, Regular
• Color: #757575
• Usage: Timestamps, subtitles
```
